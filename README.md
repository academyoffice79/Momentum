# Momentum

Momentum is a local-first productivity platform that connects projects, milestones, tasks, and calendar time blocks.

## Run

```powershell
& "C:\Users\ssvrm\.cache\codex-runtimes\codex-primary-runtime\dependencies\node\bin\node.exe" server.mjs
```

Then open:

```text
http://localhost:4173
```

## Included

- Dashboard with project health, workload, upcoming deadlines, and today's schedule
- Project management with milestones, progress, health scores, timeline summaries, and forecasts
- Task management with priority, estimates, actual time, dependencies, tags, deadlines, and status
- Weekly and monthly calendar views with task and personal event blocks
- Weekly Planning mode for dragging unscheduled tasks into calendar slots
- Daily Focus mode for today's commitments and work-session logging
- Analytics for capacity, estimation bias, deadline risk, and completion forecasts
- LocalStorage persistence with seeded demo data
